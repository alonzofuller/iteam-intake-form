TROUBLESHOOTING
================================================================================

Q: Can't find "uploading an existing file" link?
A: Click the green "Add file" button, then choose "Upload files"

Q: Drag and drop not working?
A: Click "choose your files" link and use the file picker instead

Q: Only seeing individual files, not folders?
A: That's OK! Upload the files individually:
   - index.html
   - submission-success.html
   Then create folders and upload:
   - Create "css" folder, upload styles.css into it
   - Create "js" folder, upload form.js into it

Q: GitHub Pages not working?
A: Wait 2-5 minutes after enabling it, then try again
   Make sure you selected "main" branch and "/ (root)" folder

Q: Form loads but looks ugly?
A: Make sure css/styles.css was uploaded to a "css" folder
   Make sure js/form.js was uploaded to a "js" folder

Q: Form doesn't submit?
A: Check that Formspree confirmation email and click the link (first time only)
